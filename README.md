A simple **Electronic Voting (E-Voting) System** developed as part of a machine test. The application allows voters to register, log in, cast votes, and enables an admin to view election results.
The UI is built using **AngularJS and Bootstrap**, while the backend uses **PHP and MySQL**.


## Features

### Voter
- User registration with validations
- Secure login
- View list of candidates
- Cast vote (one vote per user)


###  Admin
- Admin login
- View election results
- Clear tabular display of votes per candidate


##  Technologies Used
- **Frontend:** AngularJS, HTML5, Bootstrap 5
- **Backend:** PHP (REST APIs)
- **Database:** MySQL
- **Server:** XAMPP (Apache & MySQL)

## Setup Instructions
### Prerequisites
- XAMPP installed
- Git installed
- Web browser
